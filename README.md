## command

```shell
#推送到当前库
git push origin master 
#同步远程库
git pull origin master 
# 查看仓库连接信息
git remote -v
#冲突合并
git pull --rebase origin master
#查看远程分支
git branch -a
#查看本地分支
git branch 
#切换分支
git checkout <branch> 
```
